# 🧑‍🚀 GalaxyTours - Agência de Turismo Espacial

Este é um projeto desenvolvido para a 3º avaliação formativa de Front-End, focado em criar um site responsivo de uma agência espacial.

## 🛠️ Tecnologias Utilizadas
* HTML5 (Semântica e Acessibilidade)
* CSS3 (Estilização e Feedback Visual)
* JavaScript (Manipulação de DOM e Eventos)

## 📁 Estrutura do Projeto
O projeto foi organizado seguindo boas práticas de mercado:

/
├── index.html
└── src/
    ├── styles/  (Estilos da página)
    ├── scripts/ (Lógica de interação)
    └── images/  (Ativos visuais)
└── README.md

## 🚀 Funcionalidades Implementadas
* **Ponto de Quebra para Tablets:** Criação de regra @media para telas de 768px, com adaptação de colunas.
* **Ponto de Quebra para Smartphones:** Criação de regra @media para telas de 480px, com adaptação de colunas, estrutura flexível e definição de propriedades (Mensagem especial para usuários de computadores).
